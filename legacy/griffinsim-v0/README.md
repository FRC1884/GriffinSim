# Legacy GriffinSim v0 archive

This directory contains the pre-rebuild GriffinSim implementation that shipped versions `0.1.x`.

It is preserved strictly as an audit artifact and migration reference. It is **not** part of the active build, and new code must not depend on it.

Archived contents include:

- Maple-backed terrain-aware simulation adapters
- experimental native 6DOF classes that still depended on wall-clock stepping
- season-specific field models and release artifacts (`maven-repo/`, `vendordeps/`)
- legacy roadmap and integration documents that no longer describe the active architecture
